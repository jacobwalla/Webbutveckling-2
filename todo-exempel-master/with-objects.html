Hur fungerar det?

Istället för att bara lagra en string i ett array för varje todo lagrar vi nu ett helt objekt. En av objektets properties är ett unikt id som vi kan använda oss av när vi vill ta bort objekt ur arrayet. För att generera detta id så använder vi Date objektet. Ur det får vi en string som representerar antalet millisekunder som för förflutit sedan den 1:a januari 1970. Detta är inte den enda möjliga metoden -- unika id:n kan skapas på många olika sätt, men att använda Date är en av de smidigaste.
